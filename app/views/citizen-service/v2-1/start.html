Before you start


About local land charges

Local land charges (LLC) are legal rights and restrictions on land. For example, a tree preservation order is a local land charge that stops you from cutting down a tree.


Why you need to search for local land charges

If you are thinking about buying a house, you need to find out if it has any local land charges on it. This is because local land charges affect what you can do on a property.

This search is usually done by a conveyancer, but HM Land Registry also offers a search of the local land charges register for free. Use the free search to find out more about the property, to help you decide if you want to buy it.
