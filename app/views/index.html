Local land charges Prototypes

Version 4.1

Register a local land charge

Changes to v4 will be listed here:

  • Service name
    • Service name changed from "Local land charges register" to "Register or update a local land charge"
  • General
    • <h1> tags are updated on each page so that they now better relfect the content of the page (e.g. "Instrument" became "Charge type")
    • The type of charge page ("Instrument") has been moved to after the sign-in page.
    • Explanatory texts on various pages have been updated for clarity
  • Searching for a place
    • If users search by place name, they are taken to a map where they can draw the boundaries
    • If users search by postcode, they are taken to a page that lists the addresses. After they select an address, they are taken to a page with a map that highlights the address

Version 4

Register a local land charge

  • General
    • Added gov.uk header
  • Charge description
    • There are 2 UI versions to select the charge type - one where it uses accordion columns and one where it uses autocomplete.
    • There are also variations to these versions: one where it shows how many legislations apply to charge type (accordion) and one where there is a link to a gov.uk page where it explains what these charge types are (autocomplete).
